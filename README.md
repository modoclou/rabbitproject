### 주요 기능

#### MBTI별 맞춤 영화 추천
자신의 MBTI를 입력하면 OpenAI가 해당 MBTI의 잘 알려진 성격상을 파악하여, 추천 영화 5편을 큐레이팅

#### 사용자 접근성
가입 없이도 간편하게 이용할 수 있는 오픈 액세스 사이트

#### 추천 결과 UI
포스터 사진에 마우스를 호버링하면 TMDB API로 데이터베이스에서 가져온 해당 영화 정보를 표시

#### 회원가입 / 로그인
자유로운 가입 가능, 토큰 기반 로그인 및 비밀번호 찾기
