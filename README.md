# 🎬 MBTI 기반 영화 추천 웹사이트
Cinemine은 Cinema + mine(나의) 의 합성어로, 내 손 안의 작은 영화관을 뜻합니다.</br>
Cinemine은 MBTI와 성격 특징 기반으로 OpenAI가 영화를 선별하여 소개해 주는 사이트로, 취향에 맞는 숨은 영화를 찾을 수 있습니다.
영화를 잘 모르는 사람도 쉽게 이용할 수 있는 사이트 제작을 목표로 했습니다.

### 🎬 주요 기능
#### 사용자 접근성
가입 없이도 간편하게 이용할 수 있는 오픈 액세스 사이트
